SASS

// Syntactically Awesome Style Sheets

SASS or Syntactically Awesome Style Sheets is a language extension of CSS. SASS is more intuitive and easier to read than CSS, as it mimics the natural hierarchical structure of HTML tags and does not require curly brackets{} or semi-colons;.
Some features of SASS:
  • Can use modules to organize your style guide, making it easier to maintain and update
  • Handle repeated elements with ease using variables for elements and mixins for formats
  • Complies into any version of CSS for use with your website
